# PHP CRUD Application with User Authentication

This is a simple PHP application demonstrating basic Create, Read, Update, and Delete operations along with user registration, login, and session management.

## Setup Instructions

1.  Ensure you have a local development environment (like XAMPP) running Apache and MySQL.
2.  Clone or place the project files in your web server's document root (e.g., `htdocs` for XAMPP).
3.  Execute the `setup.sql` script in your MySQL database to create the `blog` database and required tables (`users` and `posts`).
4.  Configure the database connection settings in `config.php` if necessary. (Default is `localhost`, `root`, no password).
5.  Access the application via your browser at `http://localhost/MYPROJECT/register.php`.
